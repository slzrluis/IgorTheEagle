## Install Igor in 3 Easy Steps!

### Step 1: Download the Extension
- You should have received a folder called `igor-assistant-extension`
- Keep it somewhere safe on your computer (e.g., Documents folder)
- Don't delete this folder while using the extension!

### Step 2: Open Chrome Extensions
1. Open Google Chrome
2. Type `chrome://extensions/` in the address bar and press Enter
3. Turn ON "Developer mode" (toggle switch in top-right corner)

### Step 3: Load Igor
1. Click the "Load unpacked" button
2. Navigate to and select the `igor-assistant-extension` folder
3. Click "Select Folder"

## Uninstalling Igor

If you need to remove Igor:
1. Go to `chrome://extensions/`
